About InstaList

InstaList is a little checklist making tool that allows you to quickly create custom, printer-friendly checklists from a web browser. Great for quick to-do lists, grocery lists, or any other disposable checklist you may need.

No data is sent to any server. To save a list to your local computer, use the 'Export' button. To load a saved list, use the 'Import' button.
